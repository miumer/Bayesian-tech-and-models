Esimene samm on kirjutada funktsioone, mis simuleerivad tÃ¤ielikke tingimuslikke jaotusi, mille tuletasime eelmises osas (Lesson 5.2). Nende funktsioonidega me paneme paika tÃµmbasmise jaotuste (tÃ¤ielikud tingimuslikud jaotused) parameetrid ja seejÃ¤rel tÃµmbame vÃ¤Ã¤rtuse nendest jaotustest, mis on defineeritud Ã¼helt pool eelnevalt simuleeritud teise parameetri poolt, mida me hetkel ei simuleeri. \(\mu\) tÃ¤ielik tingimuslik jaotus teadaoleva \(\sigma^2\) ja andmete puhul on:

#siin on mu_0 ja sig2_0 eeljaotuse parameetrid
update_mu = function(n, ybar, sig2, mu_0, sig2_0) {
  sig2_1 = 1.0 / (n / sig2 + 1.0 / sig2_0)
  mu_1 = sig2_1 * (n * ybar / sig2 + mu_0 / sig2_0)
  rnorm(n=1, mean=mu_1, sd=sqrt(sig2_1))
}

tÃ¤ielik tingimuslik jaotus \(\sigma^2\) jaoks kui \(\mu\) on paika pandud ja funktsioon on

\(\text{IG}\left( \sigma^2 \mid \nu_0 + \frac{n}{2}, \, \beta_0 + \frac{\sum_{i=1}^n (y_i - \mu)^2}{2} \right)\)

#siin all on nu_0 ja beta_0 eeljaotuse parameetrid, mille me paika paneme eelnevalt
update_sig2 = function(n, y, mu, nu_0, beta_0) {
  nu_1 = nu_0 + n / 2.0 
  sumsq = sum( (y - mu)^2 ) # vectorized
  beta_1 = beta_0 + sumsq / 2.0
  out_gamma = rgamma(n=1, shape=nu_1, rate=beta_1) # rate for gamma is shape for inv-gamma kuna r ei vÃµimalda otse gammast tÃµmmata tÃµmbame algusest gammast ja siis vÃµtame inverse'i, mis on sama kui otse gammast tÃµmmata.
